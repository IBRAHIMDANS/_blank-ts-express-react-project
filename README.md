# \_blank-ts-express-react-project

This is a blank pattern of TypeScript Express (server) React (app) project to be used as an already setupped and configured project.

## Client scripts

- `yarn install`
- `yarn build`
- `yarn test`
- `yarn start`

## Server scripts

- `yarn install`
- `yarn lint`
- `yarn test`
- `yarn dev`
- `yarn build`
